Final Year University Project - Educational Piano Application

A responsive educational tool to teach users how to read sheet music along with playing piano.

The initial concept was to be able to dynamically adjust the difficulty based on the players skill. The more consecutive notes the player hits correctly, the faster the difficulty level will progress. If the player misses consecutive notes, the difficulty will decrease.

Each new level brings new challenges such as introducing more notes on a scale, increasing speed, introducing more scales and introducing chords.

The outcome from this project was to be able to produce data that could be visually represented in order to show the progress of the player. This would be used to show the player how they are improving and what areas they need to work on along with how much time users spent on each level.

You can view the full github repository [here](https://github.com/KieranBest/Final-Year-Project).
